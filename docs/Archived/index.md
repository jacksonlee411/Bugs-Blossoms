# Archived

本目录用于归档已完成的历史文档（决策快照/阶段性总结等），以保留上下文与决策依据。

## 约定

- 归档文档不再作为“活体指南”维护；如需长期维护的内容，请迁移到 `docs/guides/` 或 `docs/ARCHITECTURE.md`。
- 归档文档标题或文件头部应标注 `[Archived]`，并尽量注明归档时间与原因（例如：被更可维护的活体文档替代）。
- 禁止在 `AGENTS.md` / `CONTRIBUTING.MD` / `README.MD` 中把归档文档当作 SSOT 引用；最多作为“历史参考”链接。
